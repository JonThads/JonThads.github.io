Project Overview

This comprehensive API testing framework was developed using Postman to ensure robust validation of REST APIs using Fake APIs from JSON Placeholder.

Technologies & Tools

Postman
API testing platform for request management and test automation
JSON Schema
Response validation and data structure verification
Environment Variables
Multi-environment configuration management

Key Features

  • Comprehensive test suite covering GET, POST, PUT, PATCH, DELETE operations
  • Dynamic environment variable management for Local

Challenges & Solutions

Challenge: URL Management for GET and POST, and PATCH
Solution: API URLs for GET pertains to the general items of the API while POST and PATCH refers to each individual items.
